# Ecommerce-PowerBI-Dashboard
This project showcases an end-to-end Power BI dashboard for e-commerce sales analysis across India. It covers data cleaning, modeling, DAX measures, and interactive visualizations such as maps, bar charts, and slicers to help stakeholders monitor sales, profit, and category performance.
#  E-commerce Sales Dashboard | Power BI

##  Project Overview
This project is a Power BI dashboard created to analyze and track online sales performance across India for **Madhav Store**.

The dashboard helps stakeholders understand:
- Sales trends
- Profitability
- Category-wise and state-wise performance
- Customer purchasing behavior

---

##  Objective
To build an interactive dashboard that enables data-driven decision-making by visualizing key e-commerce metrics.

---

##  Dashboard Features
- Total Sales, Profit, Quantity KPIs
- State-wise sales analysis (Map visualization)
- Category & sub-category performance
- Monthly sales trend analysis
- Interactive filters and slicers

---

##  Tools & Technologies
- Power BI
- DAX
- Excel / CSV
- Data Modeling

---

##  Visualizations Used
- Bar Chart
- Clustered Bar Chart
- Pie & Donut Chart
- Line & Area Chart
- Scatter Plot
- Map
- Slicers
